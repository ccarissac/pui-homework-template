The (Forgotten) Memories Box

Welcome to the box of
precious memories.

This box is not designed to help locate memories, but is created for those little moments of fond rememberance. I purposely included more obscure, but still precious, memories in this project.

To interact with this project click on different parts of the smoke, or press "m" on the keyboard, and a memory will pop up. Press esc to exit the memory.

I hope you carry these moments with you.
